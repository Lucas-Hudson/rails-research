# Rails search page

Simple search page with filters and search by location.

## Getting Started

1 - Bundle install
```
$ gem install bundler # If you don't have bundler 2
$ bundle install
```

2 - Install webpacker
```
$ bundle exec rails webpacker:install
```

3 - Setup database
```
$ rails db:create 
$ rails db:migrate
$ rails db:seed
```

4 - Start server
```
$ rails s
```

5 - Lauch site

http://localhost:3000/

## Tests

```
$ rspec
```
